# Platform-Specific Creator Vetting

Per-platform creator playbooks are inputs to candidate vetting. Read the playbook for each platform in scope before screening, then apply its signals during step 3 (Initial Screening) and step 4 (Build Influencer Profiles).

| Platform | Playbook | Use it to vet |
|----------|----------|---------------|
| X / Twitter | [../../../references/platforms/x.md](../../../../references/platforms/x.md) | reply-guy vs. authority signals, thread engagement, bot-follower patterns |
| LinkedIn | [../../../references/platforms/linkedin.md](../../../../references/platforms/linkedin.md) | B2B authority, dwell/comment quality, employer/role authenticity |
| TikTok | [../../../references/platforms/tiktok.md](../../../../references/platforms/tiktok.md) | view-to-follower ratio, sound/trend fit, comment authenticity |
| YouTube | [../../../references/platforms/youtube.md](../../../../references/platforms/youtube.md) | watch-through, sub-to-view ratio, sponsorship saturation |
| Reddit | [../../../references/platforms/reddit.md](../../../../references/platforms/reddit.md) | karma/account age, subreddit standing, self-promo tolerance |

Each playbook lists platform-native metrics, red flags, and authenticity checks. Map its signals to the per-influencer profile fields (Metrics, Audience Quality Score, Brand Fit) and to the red-flag screen.
